# レスポンシブデザイン仕様

## ブレークポイント定義

| 名称 | 幅 |
|------|-----|
| モバイル | < 640px |
| タブレット | 640px - 1023px |
| デスクトップ | >= 1024px |

## コンポーネント別サイズ仕様

### page.tsx (コンテナ)

| 要素 | モバイル | タブレット | デスクトップ |
|------|----------|------------|--------------|
| padding-x | px-4 (16px) | px-6 (24px) | px-6 (24px) |

### PortfolioChart.tsx (ドーナツチャート)

| 要素 | モバイル | タブレット | デスクトップ |
|------|----------|------------|--------------|
| チャート高さ | h-[300px] | h-[380px] | h-[450px] |
| innerRadius | 90px | 110px | 130px |
| outerRadius | 130px | 160px | 190px |
| 中央ラベル(資産総額) | text-2xl | text-3xl | text-4xl |
| 中央ラベル(損益額) | text-lg | text-xl | text-2xl |
| 中央ラベル(損益率) | text-base | text-lg | text-xl |

### AssetCard.tsx (銘柄カード)

| 要素 | モバイル | タブレット | デスクトップ |
|------|----------|------------|--------------|
| padding | px-4 py-4 | px-5 py-5 | px-5 py-5 |
| アイコンサイズ | h-10 w-10 | h-12 w-12 | h-12 w-12 |

## Tailwind CSSブレークポイント

```
sm: 640px  (タブレット開始)
lg: 1024px (デスクトップ開始)
```

## 注意事項

- ResponsiveContainerの問題: ビューポートリサイズ時にwidth/height=-1になる場合がある
- ドーナツ中央ラベルエリアのサイズもinnerRadiusに合わせて調整が必要
